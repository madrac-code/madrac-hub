# Phase 1 — Runtime Foundation (STUB)

**Status**: NOT STARTED  
**Prerequisite**: Phase 0 complete  
**Goal**: Implement the first real integration between two components

## Planned Scope

The first integration will be the most concrete use case: SUBS → DUBS ("Dub Now" button).

This single use case will drive the actual Event Bus design. Architecture will emerge from implementation, not the other way around.

## Principle

Do not implement MADRAC-CORE from the 757-line design document. Implement the SUBS→DUBS integration first. Extract the abstraction from the concrete implementation. Then generalize to the Event Bus.
