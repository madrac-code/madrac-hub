# PROYECTO MADRAC: ÍNDICE DE INVESTIGACIÓN HISTÓRICA

## Visión General del Proyecto

Este proyecto documenta la historia completa del desarrollo del ecosistema MADRAC desde sus raíces iniciales hasta su estado actual (junio 2026). La investigación abarca tres componentes principales:

- **MADRAC-SUBS** (motor de subtitulación y segmentación)
- **MADRAC-ASISTENTE** (asistente JARVIS con capacidades multi-modelo IA)
- **MADRAC-DUBS** (motor de doblaje con Edge TTS)

Complementa este ecosistema el **MADRAC-HUB** (coordinador central del ecosistema).

El proyecto evolucionó desde un simple asistente JARVIS para Windows hacia un ecosistema completo de aplicaciones especializadas que colaboran mediante un Event Bus + IPC Layer compartido.
